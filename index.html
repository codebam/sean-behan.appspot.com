Flask webserver

Python webserver using the Flask web framework.

Flask webserver is a webserver written in Python using the Flask web framework that hosts web content in the current folder like a regular web server would. Instead of having to move files all over.

This makes it compatible with Google app engine as well as other python based web hosts.
